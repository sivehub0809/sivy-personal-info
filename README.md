# Eang Siv Y Personal Website

This is a static personal website ready to be hosted on GitHub Pages.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

## Publish To GitHub Pages

1. Create a new GitHub repository.
2. Upload the files in this folder to the repository root.
3. Push the repository to GitHub.
4. In GitHub, open the repository settings.
5. Open the `Pages` section.
6. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
7. Save the settings.

Your site will be published at:

- `https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/`

If you want the site at:

- `https://YOUR_GITHUB_USERNAME.github.io/`

then the repository name must be:

- `YOUR_GITHUB_USERNAME.github.io`
